# Classificação de Grãos de Café com Redes Neurais Convolucionais (ResNet50)

## Integrante
**Nome completo:** Augusto Baroni Simionato

## Descrição dos Descritores Implementados
- **ResNet50 Descriptor:** Este descritor utiliza a arquitetura ResNet50 pré-treinada para extrair características visuais relevantes das imagens. As camadas convolucionais profundas capturam padrões de textura, formas, bordas e detalhes estruturais que são úteis para a classificação dos grãos de café.

## Classificador e Acurácia
- **Classificador Utilizado:** Modelo baseado em ResNet50 com fine-tuning para adaptação ao dataset.
- **Acurácia Obtida:** A acurácia final do modelo variou entre 82% a 83% ao realizar o teste de classificação.
